### Examples

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X POST \
      'https://api.usecortex.ai/delete/delete-sources' \
      -H 'Authorization: Bearer YOUR_API_KEY' \
      -H 'Content-Type: application/json' \
      -d '{
        "tenant_id": "tenant_1234",
        "sub_tenant_id": "",
        "ids": ["CortexDoc1234", "CortexDoc4567"]
      }'
    ```

## Request Body

| Field           | Type             | Required | Description                                                                                                                 |
| --------------- | ---------------- | -------- | --------------------------------------------------------------------------------------------------------------------------- |
| `tenant_id`     | string           | Yes      | Your tenant identifier                                                                                                      |
| `sub_tenant_id` | string           | No       | Sub-tenant identifier; omit or use `""` for the default sub-tenant                                                          |
| `ids`           | array of strings | Yes      | Source IDs to delete (e.g. from upload response or [List Sources](/api-reference/endpoint/list)). At least one ID required. |
