## List memories by ID 

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X 'POST' \
    'https://api.usecortex.ai/list/list_memories' \
    -H 'accept: application/json' \
    -H 'Content-Type: application/json' \
    -d '{
    "ids": [
    "string"
    ],
    "tenant_id": "string",
    "sub_tenant_id": "string"
    }'
    ```
  </Tab>
</Tabs>

## List knowledge by ID

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X 'POST' \
    'https://api.usecortex.ai/list/list_knowledge_by_id' \
    -H 'accept: application/json' \
    -H 'Content-Type: application/json' \
    -d '{
    "ids": [
    "string"
    ],
    "tenant_id": "string",
    "sub_tenant_id": "string"
    }'
    ```
  </Tab>
  <Tab title="TypeScript">
    ```ts
    const specificSources = await client.sources.getByIds({
      tenant_id: "tenant_1234",
      sub_tenant_id: "sub_tenant_4567",
      ids: ["doc_123", "doc_456"]
    });
    ```
  </Tab>
  <Tab title="Python (Sync)">
