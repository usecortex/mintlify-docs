---
title: "List"
description: "Retrieve all memories or knowledge for a specific tenant and subtenant combination (POST /list/data)"
openapi: "POST /list/data"
---

Use **POST /list/data** with a JSON body to list either knowledge sources or user memories. Set `kind` to `"knowledge"` or `"memories"` and include `tenant_id` (and optionally `sub_tenant_id`).

### List memories

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X 'POST' \
    'https://api.usecortex.ai/list/data' \
    -H 'accept: application/json' \
    -H 'Content-Type: application/json' \
    -d '{"tenant_id": "tenant_123", "sub_tenant_id": "sub_tenant_456", "kind": "memories"}'
    ```
  </Tab>
  <Tab title="TypeScript">
    ```ts
    const allSources = await client.list.data({
      tenant_id: "tenant_1234",
      sub_tenant_id: "sub_tenant_4567"
    });
    ```
  </Tab>
  <Tab title="Python (Sync)">
    ```python
    # Async usage is similar, just use async_client and await
    all_sources = client.list.data(
        tenant_id="tenant_1234",
        sub_tenant_id="sub_tenant_4567"
    )
    ```
  </Tab>
</Tabs>
