---
title: "List"
description: "Retrieve all memories or knowledge for a specific tenant and subtenant combination"
openapi: "GET /list/list-sources"
---

### List memories

<Tabs>
  <Tab title="API Request">
    ```bash
    curl -X 'GET' \
    'https://api.usecortex.ai/list/list_memories?tenant_id=tenant_123' \
    -H 'accept: application/json'
    ```
  </Tab>
  <Tab title="TypeScript">
    ```ts
    const allSources = await client.memories.getAll({
      tenant_id: "tenant_1234",
      sub_tenant_id: "sub_tenant_4567"
    });
    ```
  </Tab>
  <Tab title="Python (Sync)">
    ```python
    # Async usage is similar, just use async_client and await
    all_sources = client.memories.get_all(
        tenant_id="tenant_1234",
        sub_tenant_id="sub_tenant_4567"
    )
    ```
  </Tab>
</Tabs>
