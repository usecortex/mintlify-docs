## Parameters

| Parameter       | Type     | Required | Description                                                                                        |
| --------------- | -------- | -------- | -------------------------------------------------------------------------------------------------- |
| `tenant_id`     | string   | Yes      | Primary organizational identifier                                                                  |
| `sub_tenant_id` | string   | Yes      | Sub-tenant ID; use empty string for default sub-tenant                                              |
| `ids`    | string[] | Yes      | List of source IDs to delete                                                                       |
